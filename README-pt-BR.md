# GtkSharp_Calculator

Aplicativo para cálculos simples de expressões aritméticas.</br>
Desenvolvido em <b>GTK# 2.0</b> e <b>Mono</b>.</br>

## IDE
	
- <b>MonoDevelop</b>
- Mono 6.10.0.104

## Teste

- Linux
- Debian 9
- OpenBox

## Requerimentos

- NUnit.3.12.0
